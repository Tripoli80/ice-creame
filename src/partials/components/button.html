Зразок кнопок для тегу button

Щоб використати колір і анімацію ховера додайте клас button--red чи button--white

Для задавання margin, pading, border-radius, font-size, line-height та іншого використовуйте унікальний клас для вашої секції, наприклад about__button

Також доступний плейсхолдер саме для кнопки (не лінка) 

                %button{
                font-family: inherit;
                border-radius: 22px;
                padding: 12px 37px 11px 37px;
                }
Щоб використати - вставте @extend %button;

Щоб отримати кнопки зі стрілками додайте всередину button тег SVG з атрибутами class="button__icon" width="7px" height="10px" та use з посиланням

use href="./images/icons.svg#icon-red-button"

або

use href="./images/icons.svg#icon-white-button"
