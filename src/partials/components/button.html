Зразок кнопок для тегу button, special для команди

Щоб використати оформлення додайте кнопці клас "red-button" або "white-button"

Щоб отримати кнопки зі стрілками додайте всередину button тег SVG з класом "button__icon" та use з посиланням

use href="./images/icons.svg#icon-red-button"

use href="./images/icons.svg#icon-white-button"

Щоб задати свої налаштування (наприклад щоб перебити pading) - додайте в тег button свій унікальний клас, до прикладу class="red-button about__button"
